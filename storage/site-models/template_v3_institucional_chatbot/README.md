# Template V3 - Institucional completo com chatbot

## Estrutura
- `index.html`, `sobre.html`, `contato.html`.
- `css/style.css`: base visual institucional.
- `css/kodassauro.css`: estilo oficial do chatbot Kodassauro.
- `js/main.js`: menu mobile e interacoes basicas.
- `js/kodassauro-chat.js`: comportamento do chatbot.
- `assets/`: logo, fundos, fontes e vendors locais.

## Funcionalidades
- Formulario de contato no front.
- Botao WhatsApp em todas as paginas.
- Chatbot Kodassauro flutuante integrado ao layout.

## Personalizacao
1. Atualize textos e imagens por cliente.
2. Ajuste links de atendimento (WhatsApp, email).
3. Se necessario, adapte o script do chatbot para novos canais.
